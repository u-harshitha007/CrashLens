# reports/

This folder stores written reports and summaries generated during the project.

## What goes here?

- `eda_summary.md` — Key findings from Exploratory Data Analysis
- `cleaning_notes.md` — Notes on data quality issues found and fixed
- `insights.md` — Final insights and conclusions

## Why keep reports separate?

In a professional Data Science project, findings are documented separately
from the code (notebooks). This makes it easy for team members or supervisors
to read conclusions without running any code.

> Reports will be added as we progress through the project milestones.
